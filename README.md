# Project Overview
A java application to manage food trucks operating in the Dublin Docklands area. Users can register food trucks, view available trucks, serach, and manage orders.

# Technologies Used
- Java (NetBeans IDE)
- Java Swing GUI

## Data Structures Used
- Singly Linked List (stores food trucks)
- Stack (tracks recently added trucks)
- Queue (manages customer orders)

## How to Run
1. Open the project in NetBeans
2. Run the Main class
